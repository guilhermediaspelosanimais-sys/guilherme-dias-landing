# Landing Page — Delegado Guilherme Dias

Site de pré-campanha do Delegado Guilherme Dias, pré-candidato a Deputado Federal pelo Paraná (Podemos, nº 20).

## Estrutura

```
index.html          → página principal (landing page)
privacidade.html     → política de privacidade (LGPD)
assets/img/          → imagens da landing page
```

## Pendências antes de publicar

- [ ] Substituir `COLE_AQUI_A_URL_DO_APPS_SCRIPT` em `index.html` pela URL do Google Apps Script publicado (grava os cadastros na planilha).
- [ ] Revisão jurídica da política de privacidade (`privacidade.html`) — pendente aprovação da Vanessa (compliance).
- [ ] Configurar GitHub Pages neste repositório.
- [ ] Criar arquivo `CNAME` com o domínio `delegadoguilhermedias.com` após ativar o GitHub Pages.
- [ ] Atualizar DNS no Wix para apontar para o GitHub Pages.
